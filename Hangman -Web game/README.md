# Hangman Web Application (HTML + CSS + PHP)

Simple web-based Hangman game interface with user registration, login, and ranking pages.

This project demonstrates basic full-stack web development using front-end design and server-side scripting.

### Home Page

The landing page contains:

* navigation menu
* banner with game logo
* ranking sidebar

From this page users can access ranking, registration and login.

### Login Page

The login form allows the user to enter:

* username
* password

The form validates input fields before submitting.

### Register Page

The registration page includes 5 required fields:

* First name
* Last name
* Email
* Password
* Repeat password

After filling the form, a new user account can be created.

### Ranking Page

Displays the leaderboard of players and their scores.
