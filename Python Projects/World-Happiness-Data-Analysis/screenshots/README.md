## Project Overview

This project analyzes the World Happiness Report dataset using Python.
The aim is to explore global happiness trends, compare regions, and understand factors associated with happiness levels.

The project performs statistical analysis, trend analysis, and data visualization across countries and years.

This is a practical Data Science and Exploratory Data Analysis (EDA) project based on real-world data.

---

## Dataset

The dataset (`world_happiness_report.csv`) contains:

* Country
* Region
* Year
* Happiness Score
* GDP per capita
* Health (life expectancy)
* Freedom
* Trust (government corruption)
* Generosity

The data is analyzed across multiple years to observe global and regional trends.
