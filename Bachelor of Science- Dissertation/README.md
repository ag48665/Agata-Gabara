## Project Description

This repository contains my Bachelor of Science dissertation project focused on analyzing airline customer satisfaction using real-world passenger data.

The goal of this project was to explore how flight delays, travel class, and flight distance influence passenger satisfaction and overall flight experience.

---

## Key Objectives

* Perform data cleaning and preprocessing
* Conduct exploratory data analysis (EDA)
* Identify relationships between delays and satisfaction
* Analyze the impact of passenger demographics
* Create clear visualizations for interpretation

---

## Dataset Information

The dataset includes airline passenger information such as:

* Age
* Gender
* Flight distance
* Departure delay
* Arrival delay
* Travel class (Business / Economy)
* Satisfaction rating

* Interactive Tableau dashboard

---

## Key Findings

* Flight delays strongly reduce passenger satisfaction
* Business class passengers show higher satisfaction levels
* Arrival delay is highly correlated with departure delay
* Long-distance flights are more sensitive to delays
